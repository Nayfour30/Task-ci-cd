# Task CI/CD 

Repositório criado para praticar os conceitos de **Integração Contínua (CI)** e **Entrega Contínua (CD)** utilizando **GitHub Actions**.

## Pipeline

O pipeline automatiza etapas de:
- Checkout do código
- Instalação de dependências
- Execução de testes simulados
- Implantação simulada

## Ferramenta utilizada
- **GitHub Actions**
